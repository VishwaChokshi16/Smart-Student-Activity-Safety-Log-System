# Smart Student Activity & Safety Log Automation System

## Introduction
The Smart Student Activity & Safety Log Automation System is a UNIX shell scripting project developed to automate the process of monitoring and recording student activities in an educational environment.

Manual activity tracking can be inefficient and error-prone. This system automates logging, monitoring, and safety alerts using shell scripting in a UNIX/Linux environment.

## Objectives
- Automate student activity tracking
- Maintain log files of student actions
- Monitor login activities
- Generate safety alerts
- Reduce manual record keeping

## Technologies Used
- UNIX / Linux (Kali Linux)
- Bash Shell Scripting
- Log Files
- Terminal Commands

## Project Files

student_login.sh  
Script to record student login activity.

activity_logger.sh  
Logs student activities into a log file.

safety_monitor.sh  
Monitors unusual activities and generates safety alerts.

view_logs.sh  
Displays stored activity logs.

logs/activity.log  
Stores activity records generated by the system.

## How to Run the Project

Step 1: Open terminal

Step 2: Navigate to project folder

cd Smart-Student-Activity-Safety-Log-System

Step 3: Run scripts

bash student_login.sh

bash activity_logger.sh

bash safety_monitor.sh

bash view_logs.sh
